# ローカルテスト手順

リファクタリング後のコードをローカルで確認する手順です。

## 方法1: Python簡易サーバー（推奨）

### Windows
```bash
# PowerShellを開く
# kanji-searchフォルダに移動
cd C:\Users\onosh\作業フォルダ\kanji-search

# サーバーを起動
python server.py
```

### macOS / Linux
```bash
cd ~/作業フォルダ/kanji-search
python3 server.py
```

### サーバー起動後
1. ブラウザを開く
2. 以下のURLにアクセス
   ```
   http://localhost:8000
   ```

3. サーバーを停止するには `Ctrl+C` を押す

## 方法2: Node.js http-server

Node.jsがインストールされている場合：

```bash
# http-serverをグローバルインストール（初回のみ）
npm install -g http-server

# kanji-searchフォルダで起動
cd /path/to/kanji-search
http-server
```

## 方法3: Python標準モジュール（簡易版）

```bash
cd /path/to/kanji-search
python -m http.server 8000
```

## トラブルシューティング

### サーバーが起動しない
```
Address already in use
```
→ ポート8000が既に使用されています。別のポート（8001など）を使用してください：
```
python -m http.server 8001
```

### CORSエラー
外部ファイルが読み込まれていない場合は、サーバーを使用してローカルホストでアクセスしてください。

### ファイルパスの問題
- Windowsユーザーの場合、バックスラッシュをスラッシュに変更
- 相対パスが正しいことを確認
